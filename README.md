> video-hhh

可用于统计视频时长

### 🏠 [Homepage](https://github.com/cuixiaorui/koa-setup)

## Install

```sh
npm install video-hhh -g
```

## Usage

```sh
video-hhh "video-path"

// 举个栗子
video-hhh "./example/*.mp4"
```

> video-path 支持 [glob](https://www.npmjs.com/package/glob) 格式的路径
